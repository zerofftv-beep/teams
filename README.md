# SBW Teams

Система команд для Superb Warfare.

## Особенности

- Неограниченное количество групп
- HUD с никами + HP участников (левый нижний угол)
- Видимость ников через блоки
- Привязка транспорта к группе
- **Название группы отображается над транспортом** (только для членов этой группы)

## Как работает транспорт

- Первый игрок из группы, севший в транспорт, автоматически привязывает его.
- Только члены этой группы могут садиться.
- Привязка сохраняется после смерти.
- При деактивации транспорта (монтировка) привязка снимается.
- Над транспортом видно название группы (только своим).

## Команды

### Для игроков
- `/team create <name>`
- `/team invite <player>`
- `/team join <name>`
- `/team leave`

### Для операторов
- `/teamadmin maxsize <number>`
- `/teamadmin nametagdistance <blocks>`
